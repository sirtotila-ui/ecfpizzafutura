# Portfolio Pizzeria Tradizionale

Sito web professionale per pizzeria tradizionale napoletana, sviluppato con React e Vite.

## Caratteristiche

- **Design Moderno**: Interfaccia elegante e professionale
- **Animazioni Fluide**: Animazioni con Framer Motion
- **Responsive**: Ottimizzato per tutti i dispositivi
- **SEO Ottimizzato**: Meta tags, structured data, Open Graph
- **Accessibilità**: Conforme WCAG, ARIA labels, keyboard navigation
- **Performance**: Lazy loading, code splitting, ottimizzazioni

## Tecnologie

- React 18
- Vite
- Framer Motion
- React Intersection Observer

## Struttura Progetto

```
src/
├── components/     # Componenti React
├── constants/       # Dati e configurazioni
├── hooks/          # Custom hooks
├── styles/         # Stili e tema
└── utils/          # Utility functions
```

## Installazione

```bash
npm install
```

## Sviluppo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Componenti Principali

- **Navbar**: Navigazione con scroll detection
- **Hero**: Sezione hero con animazioni e statistiche
- **Menu**: Menù interattivo con filtri e modal
- **Gallery**: Galleria foto con lightbox
- **Team**: Sezione team con foto
- **Map**: Mappa interattiva
- **Footer**: Footer con form contatti e newsletter

## Personalizzazione

Modifica i dati in `src/constants/config.js` e `src/constants/data.js` per personalizzare contenuti, contatti e informazioni della pizzeria.
